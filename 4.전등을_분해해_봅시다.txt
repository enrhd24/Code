
전기 : 컴퓨터 내부에서 어떻게 사용되는지 이해하기 위한 몇몇 기본적인 개념


손전등
: 이불 속에서 뭔가를 읽는다.
: 부호화된 메시지를 전달한다.

전기 회로 
: 원형의 순환형태
: 수도관을 통한 물이 흐르는 것.
: (전기) 전자의 움직임으로 인하여 발생된다 

스위치 : 회로에서 어떤 끊김이 있는 경우
회로에서 전기란 : 전자가 원자에서 다른 원자로 이동하는 것


원자 (전자, 양자, 중성자): 핵원자 주위를 회전하고 있는 전자들
분자 : 원자와 다른 원자들이 화학적으로 결합되어 형성하는 것

화합물 : 물, 소금
혼합물 : 물과 소금의 속성을 각각 유지하고 있는 경우
전기 : 특정한 상황에서 전자가 원자에서 떨어져 나오는 현상
// 전기와 전자 : 매우 작고 보이지 않는 것

전자와 양자의 관계 : 반대의 전하는 서로 끌어당기고, 같은 전하는서로 밀어낸다.

핵력 : 양자가 같은 전하를 가진 물질끼리 밀어내는 힘보다 
      훨씬 강력한 어떤힘에 의하여 같이 뭉쳐있게 되는 경우


전구? 전기?
: 전구가 전기의 이동을 촉진시킬 것 같이 않음
: 전기의 이동을 유발하는 것은 [전지]일 것이다.

전지
: 분자들이 분해되어 다른 분자가 된다
: 분자들이 서로 묶여 새로운 분자의 형태를 취하는 화학적 상호작용이 발생
: (-)부분에는 여유전자 생성, (+)부분에는 전자를 요구하는 형태의 상호작용 선택
// 화학에너지가 전기에너지로 변환되는 것

전기를 거꾸로된 경우
: 화학적 상호작용을 위해 양극에서 전자를 요구하지만,
  전자가 양극으로 도달할 방법이 없기 때문에 회로는 동작하지 않는다.

전지란?
: 전지에서 화학적 상호작용을 발생시킬 수 있도록 방법을 제공하기도 하는 것
: 음극에 전자를 가지고 와서 전지의 양극으로 전달하는 역할을 수행하는 것
: 화학적 상호작용이 소진될때까지 진행되며, 이경우 버리든 충전해야함.


전도체 혹은 도체
: 원자의 구조와 밀접하게 관련되 되어 있습니다.
: 전자의 궤도는 다양한 준위(전자껍질)를 가지고 있다
// 하나의 전자만이 전기가 흐르는 우선순위가 높아 전기에 필요한 것 
예시 > 구리, 은, 금

절연체
: 전기의 전달을 어렵게 하는 성질
: 어떤 물질이 매우 놓은 저항 값을 가져 전달시킬 수 없는 것

전선의 굵기옴
: 굵기가 굵어진에 따라 전선을 따라 흐를 수 있는 기능성이 있다.
: 굵기에 따라 흐를 수 있는 전자의 수가 증가한다.

전압 : 뭔가 일을 할 수 있는 잠재력
전류 : 실제 회로를 따라 빠르게 도는 전자의 개수와 관련이 있다.

// 전류 : 수고돤을 통하여 흘러가는 물의 양
   전압 : 수압과 비슷하다.
   저항 : 수고돤의 굵기와 유사점이 있다.

옴의 법칙
: 전자의 흐름을 방해하는 속성을 얼마나 가지고 있는 나타내는 저항 값
I -> 암페어 단위의 전류
R -> 저항을 표시하고 있다.
E -> 기전력을 나타내는 전압

쇼트회로 : 전지의 양극과 음극을 매우 짧은 구리선으로 연결되어 있는 경우

실제로 전류의 양이 전지의 물리적 크기에 의해 제한되어
많은 전류를 공금하지 못합으로 전압이 떨어지는 현상이 나타낸다.
-> 만일 전류를 공급할 수 있다면, 빛이나거나 녹아버릴 수 있다.

전구의 개발
: 전선의 저항은 전선을 뜨겁게 만들면 텅스템을 녹을 정도의 온도이다.
: 전구의 내부는 진공 상태로 텅스텐은 달구어져서 빛을 발산하게 된다.

전력(W)
: P = E * I
: 전구를 통하여 매초 몇개의 전자가 흐른다는 것을 의미하는 것

스위치 : 회로에 전기를 프르게 할 것이지 제어하는 역할을 수행한다.

정리
: 중간의 상태는 없지요.
: 이진코드와 간단한 전기회로의 유사점
