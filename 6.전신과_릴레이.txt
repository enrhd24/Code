전신시스템
: 전선으로 연결된 한쪽 끝에서 어떤일을 하면 
  그것이 전선의 맞은 편에 어떤 일을 유발시킬 수 있디.
: 원거리에서 손전들을 켜서 통신하기

전자석
: 강철 막대기에 댧은 전선을 수백 번감고 전선에 전류를 흐른다
: 마치 자석처럼 동작한다.

모스의 전신시스템
: 종이에 무언가를 써서 일을 수 있어야 한다는 고정 관념에 사로잡힘.
: 하지만, 인간은 게으른 사람이므로 소리만으로도 해석이 가능한 걸 알게됨

음향기가 달린 전신시스템
: 키, 음향기, 전지, 전선을 활용하여 전신시스템과 동일한 회로를 구성
: 지구는 회로의 절반을 담당하여 한가닥의 선으로도 충분하다.

다른 전신시스템
: 전선과 키만을 이용한 시스템의 경우 잡음을 견디지 못하였다.
: 장거리에 걸쳐 연결되면서 증가하는 저항의 문제이다.
: 전선이 무한정으로 늘어날 수는 없다는 건 자명합니다.

릴레이 방식(리피터 혹인 릴레이)
: 전신의 키를 두고 관리하는 사람이 수신된 메시지를 다시 전달해주는 방식
: 딸깍 거리는 소시를 적어서 수신 완료시 키를 눌려 다시 전송하는 방식
: 나무 막대기를 잉ㅇ하여 음향기 쪽에 묶어놓고, 키 쪽에 닿도록 만드는 것

스위치 : 사람의 손이 아닌 전류에 의하여 그 스위치를 제어하는 것
릴레이 : 이 장치를 이용하여 컴퓨터와 같이 대단한 것도 만들어 낼 수 있음.
