사람들의 표현 방식
: 각각 표식이 어느 쪽을 의미하더라도 별다른 차이가 없다.
: 여러가 표현 방법 중에서 적절한 방법을 선택하면 된다.
: 십진수 체계가 그다지 특별한 것이 아닌 손가락이 10개인 것이다.


비트 : 2진수를 표현하기 위해 만들어진 비트라는 단어
    : 컴퓨터 시대의 정보 처리를 위한 가장 기본적인 단위
    : 정보를 전달할 수 있는 유일한 방법이 아니다.
    : 매우 적인 정보를 전달한다는 저밍다.
    : 복잡한 정보를 전달하려면 여러 비드를 이용하면 가능하다.

롱펠로 : 의미를 전달될 수 있음을 암시적으로 이야기 한다.
통신 이론 : 여분의 정보를 추가로 사용하는 중복성이란 기번을 사용
통신 이론의 잡음 : 통신을 방해하는 모든 것을 의미한다.
예> 전화선의 전정기 -> 인간의 말은 중복성으로, 잡음이 있어도 성공적으로 이뤄진다.

폴 리비어
: 전달된 정보가 둘 이상의 가능성을 중에서 선택을 나타내고 있다.
: 우리가 사용하고 있는 단어는 사전의 있는 단어를 선택하는 것이다.
// 어떠한 정보의 가능성 중에 하나를 선택으로 단순화가 가능하면,
   그것을 비트로 표현할 수 있다는 의미이다.

글,그림,소리
: 이러한 자료로 뭔가를 표현할 수 없다는 것은
  그 정보를 비트 형태로 부호화할 수 없다는 것을 의미한다.
: 부호화하고 싶지 않은 것이다.
// 2명 모두 이 부호에 대하여 알고 있어야 의사소통이 가능하다.

비트의 의미
: 특정한 위치의 비트나 비트열이 어떤 의미를 가지는 지는 
  언제나 상황에 따라 이해되어야 한다.
// 영화나 비디오 가이드의 경우 어떻게 표현해야 하는 것에 대한 질문이다.

비트의 정의
: 부호화된 비트의 값을 받아서 2를 더하구 /2하면 별 평점 값을 얻을 수 있다.
: 우리가 부호에 의리를 부여할 때 합리적이고 일관성 있는 법칙을 사용했기 때문
: 부호의 의미를 파악하기 위해 부호 표를 숙지하고 있어야 한다.
: 비트수가 많아짐에 따라 전달할 수 있는 가능성의 수가 많아 진다.
: 개별 식별 번호를 가질 수 있고, 표현가능한값의 수는 2에 비트에 거듭제곱한 것이다.

비트의 표현의 수
: 표현하기 위해 얼마나 많은 비트를 사용할지 아는 방법
: 작업을 하기위해 밑수가 2인 로그 연산을 사용하면 된다.
: 로그 연산의 값은 항상 정수는 아니므로 낭비되는 비트를 기꺼이 감수하자.

카메라 필름
: 바둑판처럼 1~ 12까지 번호가 매겨져 있다.
: 다른 필름보다 빛에 민감하다는 의미 인 것이다.
: 필름이 빠르다는 것은 노출을 빨리해야하구, 비트로 부호화가 빠르다.
: 전기가 들어오는 것에 따라 0과1을 결정한다.

UPC
: 상품 포장에 항상 바코드로 표기되어 있는 부호
: 다양한 두께의 간격을 나뉘어져 있고, 숫자 몇 개가 적혀 있는 형태
: 다른 측면에서 이를 비트열로 표현할 수 있습니다.
: 사이트는 왼쪽[오른쪽] 보호 패턴, 가은데는 중앙 보호 페턴

패리티 확인
: 이 방식은 오류 및 일치성 확인 방식
: 위에 사례의 비트는 홀수 패리티로 왼쪽에서 오른쪽
: 위에 사례의 비트는 짝수 패리티로 오른쪽에서 왼쪽
// 일상생활에서 비트를 사용하는 것은 오류 점검을 하기 위해서 이다.

정리
: 비트를 사용하면 그림,소리,음악,영화, 평점, 사람의 의도 등등을 표현가능
: 정보가 얼마나 가능성을 가지고 있는 세어보아야 한다는 사실이다.
: 각각의 가능성에 숫자를 할당할 때, 몇 비트가 필요한지 결정하는데 이용된다.
: 비트 논리학이라는 학문은 중요한 역할을 하고 있다.
  항상 참인지 거짓인지 알아내는 것이 가장 중요한 목표이다.
  수학과 철학이 모하게 섞여 있어 0,1로 표현이 가능하다.
