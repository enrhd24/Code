8비트 데이터 
: 덧셈기, 래치, 데이터 선택기의 입력
: 해당 블록의 출력
: 스위치를 이용하여 입력되고 전구르 통하여 보이기 된다.
// 데이터 패스는 8비트의 폭을 가지고 있다.

바이트
: 비트와의 혼동을 피하기 위하여 바이트로 표기하게 되었다.
: 데이터 페스의 비트 수를 표현하기 위해 하용되었다.
: 중형 컴퓨터 시스템의 개발과정에서 8비트를 묶어서 부리는 이름으로 굳어짐.
// 4비트 단위[16]로 나누든지 2바이트[4]씩으로 나누든지 해야할 것이다.

16진수
: 1,2,3,4,5,6,7,8,9.A,B,C,D,E,F
: B6(Hex) or B6h

니블 : 바이트의 절반
밑수 : 어떤 수에 기반한 수체계인지를 알려주는 것
변환 틀 : 4자리 16진수를 10진수로 바꿀 때 사용하는 것

피제수 : 첫 번째로 사용된 수 //그 이외에는 제수라고 한다.
-> 반복해서 비제수와 제수와의 연산이 실행된다.

음수 : 2자리의 16진수는 가장 높은 자리가
      8, 9, A, B, C, E, F 로 시작된다.
