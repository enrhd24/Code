덧셈 
: 연산에 있어서 가장 기본이 되는 동작
: 컴퓨터가 수행하는 우일한 연산
// 릴레이들과 간단한 전기장치만을 이용하여 만들어 낼 것

덧셈기
: 이진수에 대래서만 동작할 수 있다.
: 키보드,화면이 제외돤 상태로 만들 것이다.
: 덧셈의 결과는 전구에 나타내게 되어 컴퓨터에서 수를 더하는 방식과 유사하다.
: 이진수를 더하는 것과 십잔수를 더하는 것은 아주 비슷하다

이진수표
: 한 비트를 두 이진수를 더하면 두비트가 된다.
: 각가의 합비트와 자리올림비트라 이야기한다.
// 위에 설명된 동작들을 수행하는 회로를 설계하면 되는 것입니다.
   설계를 간편하게 만들어주는 효과가 있어 [회로그성에 사용하는 모든 것]

제어판
: 8개의 스위치 두줄로 구성되어 있다
: 스위치는 일종의 입력장치로 덧셈을 위한 두 8비트 숫자를 입력하기 위해 사용
: 출력장치는 제어판 아래쪽에 일렬로 나열되어 있는 9개의 전구
// 다른 부분들은 논리 게이트들을 여러 가지 방법으로 서로 연결하여 구성된다.

스위치 : 논리 게이트의 릴레이를 동적시켜 적절한 전구에 불이 들어오게 만든다.
// 144개의 릴레이가 서로 연결되어 있는상태에서 이해하기 어렵기 때문에
   논리 게이트를 사용해서 이러한 문제들을 해결할 수 있다.

합(썸)
: 결과에서 오른쪽 아랫부분을 제외하면 오알 게이크의 결과가 같다.
: 왼쪽 윗부분만 제외하면 낫앤드게이드의 결과와 값이 같다.

상호배제적인 오알게이트
: 엑스오알게이트는 동작을 자세하게 설명하는 마지막 논리 게이트
: 일치 확인 게이트라고 불려, 입력이 값이 같으면 1이 된다.

반가산기
: 한 비트짜리 두 2진수를 더해서 합 비트와 바리올림 비트를 만들어 낸다.
: 이진수가 반 비트만으로 이루어진 것은 아니다.
: 아랫자리에서 덧셈을 통해 올라온 자리올림을 처리할 수 없다는 것이다.
: 오른쪽의 가장 낮은 자릿수에서만 사용할 수 있습니다.
  2진수의 덧셈은 이전 자릿수에서 올라온 자리올림수를 같이 포함하여 고려되어야만 한다.
// 2개의 이진수를 더하기 위해 2개의 반가산기와 오알게이트를
   아래와 같이 연결하여 사용할 수 있습니다.

1. 반가산기 : 합비트와 자리올림 비트이다.
2. 반가산기 : 첫번째 반가산기의 올라운 자리올림비트의 입력값이다.
// 2개의 반가산기아 모두 1인 경우가 없기 때문에 오알게이트만으로 사용가능하다.
   엑스오알 게이트를 사용하는 경우는 모두 1인 경우인 것이다.

전가산기 : 2개의 반가산기를 한번에 그릴 수 있는 네모 형태로 표현한다.

AND, OR, NAND 게이트는 각각 릴레이가 2개씩 사용된다.
XOr게이트는 6개의 릴레이로 구성되어 있다.
반가산기 : 엑스오알케이스와 앤드게이트로 8개의 릴레이가 사용된다.
전가산기 : 2개의 반가산기와 오알게이트 한개가 사용되고 있어 
// 모두 18개의 릴레이가 사용될 것이다.

정리 : 첫번째로 전가산기를 사용하도 자리올림 입력은 항상 0값을 
      가지도록 접지해 두어야 한다.
    : 하지만, 자리 덧셈에서도 자리 올림은 발생되어, 위자리에 입력된다.
    : 위자리에 사용되는 2개의 이진구와 전수는 전가산기에 연결된다.

축약된 내모 상자
: 전가산기에서의 자리올림 출력이 그 윗자리의 자리올림 입력을 제공된다.
   8개의 전가산기를 하나로 묶어 표현할 수 있습니다.
: 각각의 비트를 나타내기 위해서 일반적으로 사용되는 방식이다.
: 가장 오른쪽에 있는 비트들을 의미한다.
// 각 자리에 어떻게 어떻게 아래 첨자가 적용되는지 살펴조도록 하자.
: 2에 거즙제곱되는 지수에 해당되어 자릿수가 올라감에 따라 숫자가 증가된다.

8비트 가산기 : 8개의 신호가 하나로 묶여 있다는 의미 -> 8비트 숫자를 의미
16 비트 가산기 
: 8비트 덧셈기 2개를 직렬로 배치하면 만들 수 있음
: 이것도 역시 2갸의 전가산기 사이에 자리올림 형태로 되어 있음

정리
1. 덧셈기는 우리가 만든 것보다 훨씬 더 빠르게 동적하도록 만들 수 있다.
// 자리올림이 만들어진 상태에서 다음 자리 덧셈을 할 때, 입력되어야 한다.

자리올림 전파 덧셈기
: 전가산기의 처리 속도와 비트 수를 곱한 결과
: 자리올림을 예상하는 기능을 수행하는 추가회로를 이용해 속도를 높인다.

2. 컴퓨터는 더이상 릴레이를 사용하지 않는다.
: 트랜지스터는 기본적으로 릴레이와 동작은 같지만, 훨씬 상위호환이다.
// 8비트 덧셈기를 만드려면 144개의 트랜지스터가 필요하지만 
   회로는 현미겨으로 볼 수 있을 정도로 작다.
