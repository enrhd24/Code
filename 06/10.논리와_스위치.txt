진리 : 논리가 이러한 진리에 대하여 다루는 것
논리 : 진리를 찾아내기 위해서 언어를 분석한다.
논리학 
: 광범위한 내용을 담고 있는 가장 오래된 책들 중 하나
: [불:사람이름]연역법이라 불리는 삼단논법

[불의 생각]이상적인 인간
: 논리를 통하여 사고하고, 논리를 수학적으로 풀어낼 수 있음
: 인간의 뇌 활동에 애해 수학적으로 해석 가능.

대수학의 법칙 : 일반 대수학을 이용해서 계산한다.
법칙 : 모든 수학은 법칙을 기반으로 이루어져 있다.

1번째 법칙
: A + B = B + A
: A * B = B * A
2번째 법칙
: A + (B + C) = (A + B) + C
: A * (B * C) = (A * B) * C
3번째 법칙
: A * (B + C) = (A * B)+(A * C)
// 대수학의 연산자들이 숫자를 처리해야 한다
   연산자를 분리시켜서, 좀 더 추상화시켜 사용했다는 것이다.

종류[집합(set)] : 어떤 것의 묶음
부울 대수 
: 연산자는 수가 아닌 종류에 대래 적용된다.
: +, *의 기호는 완전히 다른 의미로 사용된다는 의미
  + : 두종류의 합집합 , * : 두 종류의 교집합
: 교환, 결합, 배분 법칙은 부울 대수에도 역시 적용된다.
: W + (B * F) = (W + B) * (W + F)

부울 대수 주의점
1. 2개의 기호를 더 알아야 합니다.
   2개의 기호는 숫자처럼 생겼지만 전혀 다르게 처리된다.
   2개의 기호는 절대 숫자가 아닙니다.
   // 우리가 말하는 모든 것을 의미하고 "전체 집합"이라 한다.
   ex> M + F = 1 -> '모든 종류의 고양이'
   ex> 1 - M  -> '전체에서 어떤 것을 제외하려고 할 때'
   //  1 - M = F
2. 공집합(empty set)
   : 서로 포함 관계가 없는 상호 배제적인 종류애 대해 '공집합'
   : 수컷이면서 암컷인 고양이를 찾으면 공집합이 된다.
   ex> 1 * F = F -> 모든 고양이와 암컷 고양이의 교집합
   ex> 0 * F = 0 -> 포함하지 않는 집합과 암컷 고양이의 교집합
   ex> 0 + F = F -> 포함하지 않는 집합과 암컷 고양이의 합집합
   ex> 1 + F = 1 -> 모든 고양이와 암것 고양이의 합집합
3. 부울 대수
   : 암컷 고양이와 암컷 고양이의 교집합 = 암컷 고양이
   : 일반적인 대수학과의 차이점 X^2 = X

AND and OR   
- 합집합 -> + -> OR
- 교집합 -> * -> AND
- 전체에서 뭔가를 제외하는 연산 [-1] -> NOT


컴퓨터 설계
: 조지 부울의 대수를 전자 회로와 결합시켜서 2진수 기반으로 동작하는 컴퓨터 설계
: 전구와 스위치를 연결해서 여러분이 제시한 조건을 만족시킬려고 할때,
: 여러분이 제시한 조건을 만족시키는 고양이인지 확인하는데, 도움을 줄 수 있을 까?

와(and)
: 이 회로에 전류가 흐르기 위해서 반드시 모두 닫혀 있어야만 한다.
: 손가락이 어떤 방향인지까지의 모든 정보를 2진수 혹은 비트로 표현할 수 있음.
: 어떤 스위치가 어디 있어야 하는지를 구분하지 않는다는 의미이다.
// 스위치가 빅렬로 연결되면 논리적은 앤드연산자

또는(or)
: 스위치의 위치가 바뀌더라도 결과에는 아무런 영향이 없다.
: 2개의 스위치가 병렬로 연결된 회로는 부울 대수의 OR 연산과 동일하다
// 2개의 수위치가 병렬로 연결되면 논리적인 오알연산자


정리
+ : 두 스위치가 병렬로 연결되어 있는 것
* : 두 스위치가 직렬로 연결되어 있는 것
