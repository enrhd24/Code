digit : 손가락, 발가락, five(다섯), first(주먹)과 같은 어원을 가짐.

오늘날의 수체계
: 고대 그리스인들은 대수학에 대하여 잘 알지 못한다.
: 린두-아라비아 혹은 인도-아라비아숫자로 알려져 있습니다.
// 라틴어로 번역된 로마숫자가 힌두-아라비아 숫자체계로 빠르게 변화되었다.

1. 힌두-아라비아 숫자 체계
: 숫자가 수에서 어디에 존재하는냐는 우선순위(중요도)를 나타내게 된다.
: 같은 1을 가지고 있지만, 값이 백보다 훨씬 크고 중요한 값이다.

2. 초기의 모든 숫차체계
: 십을 나타내기 위한 특별한 기호를 가지고 있었다.
: 지금도 십을 나타내기 위한 특별한 기호가 없다.

3. 초기의 없는 숫자체계
: 한두-아라비아 숫자 체계에만 존재하는 것
: 십을 우히나 기호보다 훨씬 중요한 의미인 영(0;Zero)

영(0;Zero)
: 값을 구분할 수 있는 것 <예시> 25, 205
: 연산하기 어려운 곱셉이나 나눗셈과 같은 연산을 쉽게 수행가능
: 숫자들을 발음하는 방법을 통해 힌두-아라비아 숫자의 전체구조르 알 수 있음
: 1을 다른 위치에 적고 위치를 저장하기 위해서 0을 사용하면 된다.
// 영이 없던 시절 덧셈표를 워웠어야 했던 것이었다.

: 만일 손가락이 4개였다면, 8에 기반하는 숫체계를 선호할 것이다
