사람들의 표현 방식
: 각각 표식이 어느 쪽을 의미하더라도 별다른 차이가 없다.
: 여러가 표현 방법 중에서 적절한 방법을 선택하면 된다.
: 십진수 체계가 그다지 특별한 것이 아닌 손가락이 10개인 것이다.


비트 : 2진수를 표현하기 위해 만들어진 비트라는 단어
    : 컴퓨터 시대의 정보 처리를 위한 가장 기본적인 단위
    : 정보를 전달할 수 있는 유일한 방법이 아니다.
    : 매우 적인 정보를 전달한다는 저밍다.
    : 복잡한 정보를 전달하려면 여러 비드를 이용하면 가능하다.

롱펠로 : 의미를 전달될 수 있음을 암시적으로 이야기 한다.
통신 이론 : 여분의 정보를 추가로 사용하는 중복성이란 기번을 사용
통신 이론의 잡음 : 통신을 방해하는 모든 것을 의미한다.
예> 전화선의 전정기 -> 인간의 말은 중복성으로, 잡음이 있어도 성공적으로 이뤄진다.

폴 리비어
: 전달된 정보가 둘 이상의 가능성을 중에서 선택을 나타내고 있다.
: 우리가 사용하고 있는 단어는 사전의 있는 단어를 선택하는 것이다.
// 어떠한 정보의 가능성 중에 하나를 선택으로 단순화가 가능하면,
   그것을 비트로 표현할 수 있다는 의미이다.

글,그림,소리
: 이러한 자료로 뭔가를 표현할 수 없다는 것은
  그 정보를 비트 형태로 부호화할 수 없다는 것을 의미한다.
: 부호화하고 싶지 않은 것이다.
// 2명 모두 이 부호에 대하여 알고 있어야 의사소통이 가능하다.

비트의 의미
: 특정한 위치의 비트나 비트열이 어떤 의미를 가지는 지는 
  언제나 상황에 따라 이해되어야 한다.
// 영화나 비디오 가이드의 경우 어떻게 표현해야 하는 것에 대한 질문이다.
