양방향 전신 시스템 
: 2개의 동일한 회로가 서로 연결되지 않은 독립적인 형태

공통 연결 
: 전지, 스위치, 전구, 전지가 쌍둥이처럼 연결되어 있지만, 각각 독립적으로 동작하는 것
: 회로의 공통 연결을 통해서 전류가 흐르지 않고 있다.

전선 부분
: 무생물인 금속, 돌, 물, 그리고 유기물로 이루어진 거대한 지구로 대체가능
: 전도체의 특성 중 하나는 "크면 클수록 좋다" 
: 전도체의 특성 중 하나는 "얇은 전선보다 훨씬 더 전도성이 좋다"

접지 혹은 어스
: 구리 막대를 땅에 잘 묻고 여기에 전선을 연결하는 것
: 수도관에 전선을 연결하여 공통 연결로 사용하는 방법
// 지표면에 전기적 접점을 두는 것

접지란? 
: 지표면과의 물리적 연결을 의미하는 것
: 여러분의 집과 친구의 집 사이에 오직 하나의 선만 지나가도록 만들 수 있음

전선의 지구
: 지구가 이 세상의 수많은 전자 회로들에게 동일한 일을 수행하고 있음
: 전지의 음극에서 나온 전자는 다시 땅으로 돌아가게 된다.

지구
: 전기에 대하여 대단히 큰 전도체
: 전자의 공급처이자 보관소
: 실질적으로 문한한 전자의 원천 및 전자가 플러 나갈 수 있는 매우 거대한 장소
: 지구도 어쟀든 어느 정도 저항이 있다.
: 전압이 낮은 전지를 이용하기에는 지구의 저항이 훨씬 큰 것이다.

전압 
: 진공을 상징할 수 있도 있음.
: 어떤 일을 할 수 있는 잠재력
